# Bulk TinyPesa STK Push Bot

## Install

```bash
npm install
```

## Run

```bash
npm start
```

Open:

http://localhost:3000

## Environment Variables

Create a `.env` file:

```env
TINYPESA_USERNAME=your_username
TINYPESA_API_KEY=your_api_key
PORT=3000
```

## Deploy to Render

- Push to GitHub
- Create Render Web Service
- Build command: `npm install`
- Start command: `npm start`
